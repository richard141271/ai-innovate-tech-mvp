Aktiv MVP utvikling

Lek-Bienes Vokter™

Et Hands-Free AI-helsesystem for birøktnæringen: sanntidsanalyse via bodycam og edge-AI. Birøkteren mottar umiddelbar verbal feedback om sykdom og dronningstatus, mens anonymiserte data aggregeres i skyen for prediktiv smittevarsling.

Status: Prototyping mot Innovasjon Norge-søknad. Se siste live deploy

Vår teknologi‑stack

Sanntids KI (Vision)

Pixel-for-pixel videoanalyse via OpenAI Vision / Edge AI for å detektere sykdomstegn som kan være usynlige for det blotte øye.

Hands‑Free UX

Stemmekontrollert grensesnitt og automatisk datafangst via sensorer, slik at telefonen kan bli liggende i lomma og arbeidet forblir hands‑free.

Geospatial Data

Integrasjon mot Google Maps API og Mattilsynets data for automatisert smittesporing, varsling og visualisering på kart.
