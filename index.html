Aktiv MVP Utvikling

Lek-Bienes Vokter™

Et revolusjonerende "Hands-Free" AI-helsesystem for birøktnæringen. Ved hjelp av bodycam og sanntids KI-analyse får birøkteren umiddelbar verbal feedback om sykdom og dronningstatus, mens data strømmes til skyen for prediktiv smittevarsling.

Status: Prototyping mot Innovasjon Norge søknad. (Se siste live deploy her)

Vår Teknologi-Stack

Sanntids KI (Vision)

Pixel-for-pixel videoanalyse via OpenAI Vision / Edge AI for å detektere sykdomstegn usynlige for det blotte øye.

Hands-Free UX

Stemmekontrollert grensesnitt og automatisk datafangst via sensorer, slik at telefonen kan bli liggende i lommen.

Geospatial Data

Integrasjon mot Google Maps API og Mattilsynets data for automatisert smittesporing og varsling.
